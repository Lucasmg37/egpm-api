Geral

Está API contém todos os métodos necessários para o correto funcinamento do CMS acoplado ao Site Encontro Gamer Pará de Minas. É possível realizar consultas no banco e operações de inserçoes e remoção. Todas as possibilidades de ajustes serão aqui listadas.
